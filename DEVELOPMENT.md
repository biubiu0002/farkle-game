# Farkle 骰子游戏 - 开发指南

## 项目结构

```
farkle-game/
├── components/        # Vue组件
│   └── Die.vue      # 骰子组件
├── pages/            # 页面
│   └── index/
│       └── index.vue # 主游戏页面
├── utils/            # 工具函数
│   ├── scorer.ts    # 计分逻辑
│   └── gameLogic.ts # 游戏核心逻辑
├── App.vue           # 根组件
├── main.ts           # 入口文件
├── pages.json        # 页面配置
├── manifest.json     # 应用配置
├── package.json      # 依赖配置
└── tsconfig.json     # TypeScript配置
```

## 快速开始

### 1. 安装依赖

```bash
cd farkle-game
npm install
```

### 2. 开发模式

**H5（网页）：**
```bash
npm run dev:h5
```
在浏览器打开 http://localhost:5173

**微信小程序：**
```bash
npm run dev:mp-weixin
```
使用微信开发者工具打开 `dist/dev/mp-weixin` 目录

### 3. 构建

**H5：**
```bash
npm run build:h5
```

**微信小程序：**
```bash
npm run build:mp-weixin
```

## 核心功能说明

### 游戏流程

1. **等待阶段** - 点击"开始游戏"
2. **掷骰子** - 6个骰子随机生成
3. **选择阶段** - 选择可计分的骰子
4. **继续或存分** - 决定继续掷还是存分
5. **Farkle** - 没有可计分骰子，回合结束
6. **切换玩家** - 轮到下一位

### 计分规则

| 组合 | 分数 |
|------|------|
| 1个1 | 100 |
| 1个5 | 50 |
| 三个1 | 1000 |
| 三个2 | 200 |
| 三个3 | 300 |
| 三个4 | 400 |
| 三个5 | 500 |
| 三个6 | 600 |
| 四个相同 | 1000 |
| 五个相同 | 2000 |
| 六个相同 | 3000 |
| 三对 | 1500 |
| 顺子(1-6) | 2500 |

### Hot Dice 规则

当所有骰子都能计分时，玩家可以重新掷所有6个骰子，继续积累分数。

## 扩展功能建议

1. **AI对手** - 添加智能AI对战
2. **多人在线** - 使用WebSocket实现实时对战
3. **音效和动画** - 添加掷骰子动画和音效
4. **排行榜** - 本地最高分记录
5. **自定义规则** - 允许玩家选择不同的计分规则
6. **主题切换** - 多种UI主题

## 技术栈

- **框架**: UniApp (Vue 3 + TypeScript)
- **状态管理**: 响应式数据
- **平台**: 微信小程序、H5

## 注意事项

- 骰子选择逻辑需要严格验证，避免无效选择
- Farkle检测需要准确，避免漏判
- Hot Dice状态需要正确处理骰子重置
- 微信小程序需要申请appID才能发布
